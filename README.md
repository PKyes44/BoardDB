이전에 만들었던 범죄자 조회 프로그램을 업데이트했습니다. 이전 프로그램은 C언어로 읽어들인 후 메모장에 내용을 저장하는 방법을 사용했으나, 업데이트된 프로그램은 파이썬으로 정보를 읽어들인 후에 mysql과 연동하여 데이터베이스에 저장되도록 했습니다. 업데이트로 인해 더 많은 정보와 더욱 자세하고 정확한 정보를 추출해낼 수 있습니다.

이 프로그램에서 중요한 함수는 pymysql의 함수인 connect와 excute입니다. connect는 mysql과 파이썬을 연결하는 함수이고 함수에 호스트명, 비밀번호, 추가 설정 등을 입력받습니다.
excute는 쿼리를 입력받고 mysql에 적용시키는 함수입니다.
